
**UConn Data Science & Engineering | Emotion Classification**

## Project Description
This project uses two methods Random Forest without PCA and Random Forest with PCA to compare them and see if dimensionality reduction works for emotion classification.

## Project Directory Structure
* `/data`: Local directory for raw image storage (Git-ignored).
* `/models`: Storage for serialized model weights and saved checkpoints.
* `/notebooks`: Interactive Jupyter/Colab notebooks for development.
* `/scripts`: Production-ready Python files for model execution.
